# before-submit

justify your data before form submit

## Features

- Password power meter
- Number Validation
- Object Validation
- User_Name Validation

## Installation

Install my-project with npm

```bash
 npm i before-submit

```

## Usage

```javascript
import justify from "before-submit";
```
